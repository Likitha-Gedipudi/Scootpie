# Soft Light Theme Color Guide

## Color Palette

### Primary (Soft Sage Green)
- `#9CAF9A` - Primary sage
- `#8A9D88` - Darker sage
- `#B5C4B3` - Lighter sage

### Accent (Soft Rose/Coral)
- `#E8B4B8` - Primary rose
- `#D99FA4` - Darker rose
- `#F5CDD1` - Lighter rose

### Backgrounds
- `#FAFAFA` - Main background
- `#F5F3F0` - Secondary/beige background
- `#FFFFFF` - Card background

### Text
- `#2C2C2C` - Primary text
- `#8B8B8B` - Muted text
- `#B8B8B8` - Placeholder text

### Borders
- `#E8E6E3` - Primary border
- `#F2F1EF` - Light border

### Replace These Old Colors:
- `#D4AF37`, `#B8941F`, `#E5C158` (Gold) → `#9CAF9A` (Sage) or `#E8B4B8` (Rose)
- `#000000`, `#0A0A0A`, `#1A1A1A` (Black) → `#FFFFFF` or `#FAFAFA`
- `#333333`, `#2A2A2A` (Dark gray) → `#F5F3F0` or `#E8E6E3`
- `#8B1A1A`, `#5C1010` (Burgundy) → `#9CAF9A` (Sage)
- `#DC2626`, `#A52A2A` (Red) → `#E8B4B8` or `#E39B9B` (Soft red)

## Usage Examples
- Buttons: `bg-gradient-to-r from-[#9CAF9A] to-[#8A9D88]`
- Hover states: `hover:from-[#B5C4B3]`
- Borders: `border-[#E8E6E3]`
- Text: `text-[#2C2C2C]`
- Muted text: `text-[#8B8B8B]`
- Accent elements: `text-[#E8B4B8]`
